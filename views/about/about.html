What is the Open Place Database?

A database of places, their current and historical names, and their boundaries over time.

Why is it needed?

While there are several very good sources for current places (Geonames) and boundaries (OSM), there is no global collection of historical places and boundaries with their changes over time that are available under a similar license.

What's inside?

A collection of places, each with a list of fully qualified names, the dates that the place existed, and a list of geojson objects representing the changing boundaries over time.

How does it work?

The data itself is stored as individual json and geojson files in a directory structure. This makes it easy to work with if you need to extract information. Also available is the code for this site, which includes a viewer, a map editor, and an API for editing and querying places. For more information visit our Github Repository
